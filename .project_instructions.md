# MentorConcursos - PWA Estático

## Visão Geral
PWA estático (HTML+CSS+JS puro, SEM frameworks) para gerenciamento de estudos de concursos públicos brasileiros. Hospedado no GitHub Pages em `https://github.com/rafaeaguiarecai-beep/MentorConcursos`.

## Arquitetura
- **NÃO USA Next.js/React** - apenas HTML+CSS+JS vanilla (SPA com router próprio)
- Pasta `nextjs_space/` existe mas NÃO é usada - todos os arquivos do app estão na raiz `/home/ubuntu/mentor_concursos/`
- Caminhos sempre RELATIVOS (`./`, `js/...`, `icons/...`) para GitHub Pages
- Idioma: pt-BR

## Estrutura de Arquivos
```
index.html              # SPA shell com bottom nav + main content
css/style.css           # Tema dark
js/db.js                # Dexie.js (IndexedDB) + helpers de data/tempo
js/timer.js             # Timer com Web Audio API (800Hz x3)
js/backup.js            # Export/Import/Share JSON
js/charts.js            # Chart.js (barras horizontais + linha 8 semanas)
js/app.js               # SPA router + páginas + modais
manifest.json           # PWA manifest
sw.js                   # Service Worker cache-first
icons/icon-192.png      # PNG gerado via Pillow ("MC" em accent)
icons/icon-512.png
```

## Tema (Dark)
- Background: `#1a1a2e`
- Cards: `#16213e`
- Highlight: `#0f3460`
- Accent: `#e94560`
- Border-radius: 10px (5px sm)

## Banco IndexedDB (MentorConcursosDB)
Tabelas: `concursos`, `disciplinas`, `topicos`, `sessoes`, `revisoes`, `cicloConfig`

## Páginas (SPA via Router.ir)
1. **dashboard** - countdown live, 4 cards de stats, termômetro de aprovação, gráfico de horas/disciplina, próximas 5 revisões, banner de backup (7 dias)
2. **estudar** - timer circular SVG com Web Audio alarme + vibração + notificação, conta tempo extra em verde, modal de finalizar com 5 estrelas + frase motivacional (20 frases)
3. **revisoes** - tabs Hoje/Atrasadas/Próximos7/Todas, click leva para Estudar pré-preenchido
4. **historico** - agrupado por dia, gráfico linha 8 semanas, filtros disciplina+período
5. **ciclo** - reorder ▲▼, geração automática proporcional ao peso (eliminatórias x1.5)
6. **mais** → ciclo / configuracoes / sobre
7. **configuracoes** - editar concurso, CRUD disciplinas, backup (export/import/share), limpar dados (dupla confirmação digitando "APAGAR")

## Servidor Dev
`cd /home/ubuntu/mentor_concursos && python3 -m http.server 3000` (sem Next.js, sem yarn)

## Frases Motivacionais
20 frases em pt-BR para concurseiros, exibidas após cada sessão registrada.
